SQL COUNT(*) 函数

SQL 函数

定义和用法

COUNT(*) 函数返回在给定的选择中被选的行数。

语法

SELECT COUNT(*) FROM table

例子

Name Age
Adams, John 38
Bush, George 33
Carter, Thomas 18

例子 1

本例返回 "Persons" 表中的行数：

SELECT COUNT(*) FROM Persons

结果：

3

例子 2

返回大于 20 岁的人数：

SELECT COUNT(*) FROM Persons WHERE Age>20

结果：

2
