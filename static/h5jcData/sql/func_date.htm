MySQL DATE() 函数

MySQL Date 函数

定义和用法

DATE() 函数返回日期或日期/时间表达式的日期部分。

语法

DATE(date)

date 参数是合法的日期表达式。

实例

假设我们有如下的表：

OrderId ProductName OrderDate
1 'Computer' 2008-12-29 16:25:46.635

我们使用下面的 SELECT 语句：

SELECT ProductName, DATE(OrderDate) AS OrderDate
FROM Orders
WHERE OrderId=1

结果：

ProductName OrderDate
'Computer' 2008-12-29
