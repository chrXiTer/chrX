HTML <thead> 标签的 charoff 属性

HTML <thead> 标签

实例

下面的 HTML 表格把 thead 元素中的内容与字符 "M" 向右两个字符的位置对齐：

<table border="1" width="100%">
  <thead align="char" char="M" charoff="2">
    <tr>
      <th>Month</th>
      <th>Savings</th>
    </tr>
  </thead>
    <tr>
      <td>January</td>
      <td>$100</td>
    </tr>
    <tr>
      <td>February</td>
      <td>$80</td>
    </tr>
</table>

亲自试一试

定义和用法

charoff 属性规定 thead 元素中的内容相对于由 char 属性规定的字符的对齐偏移量。

仅当 align 属性设置为 "char" 且已设置 char 属性时，才能使用 charoff 属性。

语法

<thead charoff="value">

属性值

值 描述
number

规定对齐方式。

正数规定向字符的右边对齐。

负数规定向字符的左边对齐。

延伸阅读：char 和 charoff 属性

即使是最简单的文字处理器也允许在表格中排列十进制数字中的点。直到 HTML 4.0 标准的出现，网页设计语言一直不能充分满足这方面的要求。

现在，可以使用 char 属性，来指定表格行内单元格中的一个字母，这个字母就是对齐的中心线。不需要为 char 指定值。如果你没有指定的话，默认的字符是基于语言：例如，在英语中是一个句点，而在法语中是一个逗号。如果把一个单字符指定为 char 属性的值，则可以指定不同的对齐字符。

使用 charoff 属性和一个整数值可指定每行对齐字符第一次出现的偏移量。如果一行中不包括对齐字符，应该水平平移到对齐位置的最后。

char 和 charoff 属性在 HTML 4 和 XHTML 中是新出现的属性，还没有得到任何一个流行浏览器的支持。
