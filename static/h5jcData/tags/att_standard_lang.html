HTML lang 属性

HTML 全局属性

实例

段落中的一些法文：

<p lang="fr">Ceci est un paragraphe.</p>

亲自试一试

定义和用法

lang 属性规定元素内容的语言。

提示和注释

注释：lang 属性在以下标签中无效：<base>, <br>, <frame>, <frameset>, <hr>, <iframe>, <param> 以及 <script>。

语法

<element lang="language_code">

属性值

值 描述
language_code 规定元素内容的语言代码。语言代码参考
