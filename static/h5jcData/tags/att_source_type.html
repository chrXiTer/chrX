定义和用法

type 属性规定媒体资源的 MIME 类型。

HTML 4.01 与 HTML 5 之间的差异

<source> 标签是 HTML 5 中的新标签。

语法

<source type="MIME_type">
值 描述
MIME_type

规定媒体资源的 MIME 类型。

常用的 MIME 类型：

用于视频：

  • video/ogg
  • video/mp4
  • video/webm

用于音频：

  • audio/ogg
  • audio/mpeg

请参阅 IANA MIME 类型，获取完整的标准 MIME 类型列表。
