</form>

<output form="numform" name="x" for="a b"></output>

亲自试一试

定义和用法

form 属性规定 <output> 元素所属的一个或多个表单。

HTML 4.01 与 HTML 5 之间的差异

<output> 标签是 HTML5 中的新标签。

语法

<output form="form_id">

属性值

值 描述
form_id 规定 <output> 元素所属的表单元素。该值必须是同一文档中的 <form> 元素的 id 属性。
