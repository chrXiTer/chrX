定义和用法

autofocus 属性是逻辑属性。

当设置该属性后，它规定在页面加载后文本区域自动获得焦点。

HTML 4.01 与 HTML 5 之间的差异

autofocus 属性 HTML5 中的新属性。

HTML 与 XHTML 之间的差异

在 XHTML 中，不允许属性简写，autofocus 属性必须定义为 <textarea autofocus="autofocus">。

语法

<textarea autofocus>
