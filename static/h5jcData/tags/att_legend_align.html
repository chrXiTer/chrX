HTML <legend> 标签的 align 属性

HTML <legend> 标签

实例

组合表单中的相关元素：

<form>
  <fieldset>
    <legend align="right">health information</legend>
    height: <input type="text" />
    weight: <input type="text" />
  </fieldset>
</form>

亲自试一试

定义和用法

align 属性规定 fieldset 中标题的对齐方式。

兼容性注释

在 HTML 4 中，不赞成使用 align 属性；在 XHTML Strict DTD 中，不支持该属性。

语法

<legend align="value">

属性值

值 描述
left 对标题进行左对齐。
right 对标题进行右对齐。
top 对标题进行上对齐。
bottom 对标题进行下对齐。
