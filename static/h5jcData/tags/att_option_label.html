HTML <option> 标签的 label 属性

HTML <option> 标签

实例

在 option 元素中使用 label 属性：

<select>
  <option label="Volvo">Volvo (Latin for "I roll")</option>
  <option label="Saab">Saab (Swedish Aeroplane AB)</option>
</select>

亲自试一试

定义和用法

label 属性规定更短版本的选项。

下拉列表中会显示出所规定的更短版本。

语法

<option label="value">

属性值

值 描述
text 选项的更短的版本。
