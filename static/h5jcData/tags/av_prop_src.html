HTML 5 Audio/Video DOM src 属性

HTML 5 Audio/Video DOM 参考手册

实例

改变视频的来源：

myVid=document.getElementById("video1");
myVid.src="movie.ogg";

亲自试一试

定义和用法

src 属性设置或返回音频/视频的当前来源。

语法

设置 src 属性：

audio|video.src=URL

返回 src 属性：

audio|video.src

属性值

值 描述
URL

规定音频/视频来源的 URL。

可能的值：

  • 绝对 URL - 指向另一个网站（比如 src="http://example.com/movie.ogg"）
  • 相对 URL - 指向网站内的某个文件（比如 src="/data/movie.ogg"）

返回值

类型 描述
字符串值 当前的音频/视频来源。
