HTML <td> 标签的 char 属性

HTML <td> 标签

实例

在下面的例子中，"Savings" 数据单元的内容与 "." 字符对齐：

<table border="1">
  <tr>
    <th>Month</th>
    <th>Savings</th>
  </tr>
  <tr>
    <td>January</td>
    <td align="char" char=".">$100.00</td>
  </tr>
  <tr>
    <td>February</td>
    <td align="char" char=".">$10.00</td>
  </tr>
</table>

亲自试一试

定义和用法

char 属性规定将单元格中的内容相对某个字符的对齐方式。

仅当 align 属性设置为 "char" 时，才能使用 char 属性。

char 属性的默认值是页面语言的小数点字符。

语法

<td char="value">

属性值

值 描述
character 规定将内容与之对齐的字符。

延伸阅读：char 和 charoff 属性

即使是最简单的文字处理器也允许在表格中排列十进制数字中的点。直到 HTML 4.0 标准的出现，网页设计语言一直不能充分满足这方面的要求。

现在，可以使用 char 属性，来指定表格行内单元格中的一个字母，这个字母就是对齐的中心线。不需要为 char 指定值。如果你没有指定的话，默认的字符是基于语言：例如，在英语中是一个句点，而在法语中是一个逗号。如果把一个单字符指定为 char 属性的值，则可以指定不同的对齐字符。

使用 charoff 属性和一个整数值可指定每行对齐字符第一次出现的偏移量。如果一行中不包括对齐字符，应该水平平移到对齐位置的最后。

char 和 charoff 属性在 HTML 4 和 XHTML 中是新出现的属性，还没有得到任何一个流行浏览器的支持。
