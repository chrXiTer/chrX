HTML <meter> value 属性

HTML <meter> 标签

实例

带有当前值以及 min、max、high、low 属性的测量范围：

<meter min="0" low="40" high="90" max="100" value="95"></meter>

亲自试一试

定义和用法

必需的 value 属性规定测量范围的当前的或“测量过”的值。

value 属性必须介于 min 与 max 属性值之间。

HTML 4.01 与 HTML 5 之间的差异

<meter> 标签是 HTML 5 中的新标签。

语法

<meter value="number">
值 描述
number 必需。规定测量范围的当前值，浮点数。
