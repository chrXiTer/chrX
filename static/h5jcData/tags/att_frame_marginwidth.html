HTML <frame> 标签的 marginwidth 属性

HTML <frame> 标签

实例

第一个框架的左边距和右边距是 50 像素，第二个框架的上边距和下边距是 0：

<html>

<frameset cols="50%,50%">
<frame src="frame_a.htm" marginwidth="50" />
<frame src="frame_b.htm" marginwidth="0" />
</frameset>

</html>

亲自试一试

定义和用法

marginwidth 属性规定框架内容与框架的左侧和右侧之间的高度，以像素计。

语法

<frame marginwidth="value">

属性值

值 描述
pixels 规定框架的左边距和右边距。
