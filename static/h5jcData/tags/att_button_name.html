HTML <button> 标签的 name 属性

HTML <button> 标签

实例

两个按钮带有相同的名称，在单击它们时提交不同的值：

<form action="demo_button_name.asp" method="get">
  Choose your favorite subject:
  <button name="subject" type="submit" value="HTML">HTML</button>
  <button name="subject" type="submit" value="CSS">CSS</button>
</form>

亲自试一试

定义和用法

name 属性为按钮规定名称。

name 属性用于在 JavaScript 中对元素进行引用，或者在表单提交之后，对表单数据进行引用。

提示：不同的 button 元素可以共享相同的名称。这就允许您标记带有相同名称的若干按钮，以便在表单中使用时能够提交不同的值。

语法

<button name="value">

属性值

值 描述
name 规定按钮的名称。
