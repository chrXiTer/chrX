HTML <select> 标签的 autofocus 属性

HTML <select> 标签

实例

自动获得焦点的下拉列表：

<select autofocus>
   <option value="volvo">Volvo</option>
   <option value="saab">Saab</option>
   <option value="opel">Opel</option>
   <option value="audi">Audi</option>
</select>

亲自试一试

定义和用法

autofocus 属性是逻辑属性。

当设置该属性后，它规定在页面加载后下拉列表应该自动获得焦点。

HTML 4.01 与 HTML 5 之间的差异

autofocus 属性 HTML5 中的新属性。

HTML 与 XHTML 之间的差异

在 XHTML 中，不允许属性简写，autofocus 属性必须定义为 <select autofocus="autofocus">。

语法

<select autofocus>
