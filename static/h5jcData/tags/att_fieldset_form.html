HTML 5 <fieldset> form 属性

HTML <fieldset> 标签

实例

属于 id 为 "iceform" 的表单的 fieldset：

<fieldset form="iceform">
  <legend>Personalia:</legend>
  Name: <input type="text" name="usrname"/><br />
  Email: <input type="text" name="usrmail" /><br />
  Date of birth: <input type="text" name="usrdate" />
</fieldset>

亲自试一试

定义和用法

form 属性规定 fieldset 所属的一个或多个表单。

form 属性的值必须是所属表单的 id。

如需引用一个以上的表单，请使用空格分隔的列表。

HTML 4.01 与 HTML 5 之间的差异

form 属性是 HTML5 中的新属性。

语法

<fieldset form="form_id">

属性值

值 描述
form_id fieldset 所属表单 id 的空格分隔列表。
