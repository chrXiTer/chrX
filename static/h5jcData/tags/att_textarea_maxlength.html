HTML <textarea> 标签的 maxlength 属性

HTML <textarea> 标签

实例

最大字符长度为 50 的文本区域：

<textarea maxlength="50">
 Enter text here...
</textarea>

亲自试一试

定义和用法

maxlength 属性规定文本区域的最大长度（以字符计）。

HTML 4.01 与 HTML 5 之间的差异

maxlength 属性是 <textarea> 标签在 HTML5 中的新属性。

语法

<textarea maxlength="number">

属性值

值 描述
number 文本区域中允许的最大字符数。
