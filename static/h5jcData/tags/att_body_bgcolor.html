定义和用法

bgcolor 属性规定文档的背景颜色。

兼容性注释

在 HTML 4.01 中，不赞成使用 body 元素的 bgcolor 属性；在 XHTML 1.0 Strict DTD 中，不支持 body 元素的 bgcolor 属性。

请使用 CSS 代替。

CSS 语法：<body style="background-color:#E6E6FA">

CSS 实例：设置文档的背景颜色

在我们的 CSS 教程中，您可以找到更多有关 background-color 的细节。

语法

<body bgcolor="value">

属性值

值 描述
color_name 规定颜色值为颜色名称的字体颜色（比如 "red"）。
hex_number 规定颜色值为十六进制值的字体颜色（比如 "#ff0000"）。
rgb_number 规定颜色值为 rgb 代码的字体颜色（比如 "rgb(255,0,0)"）。
