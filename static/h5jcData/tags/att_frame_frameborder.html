HTML <frame> 标签的 frameborder 属性

HTML <frame> 标签

实例

不带边框的双框架页面：

<html>

<frameset cols="50%,50%">
<frame src="frame_a.htm" frameborder="0" />
<frame src="frame_b.htm" frameborder="0" />
</frameset>

</html>

亲自试一试

定义和用法

frameborder 属性规定是否显示框架周围的边框。

提示：出于实用性方面的原因，最好不用设置该属性，请使用 CSS 来应用边框样式和颜色。

CSS 实例：frame 边框

语法

<frame frameborder="value">

属性值

值 描述
1 有边框（默认值）。
0 无边框。
