HTML 5 <button> formnovalidate 属性

HTML <button> 标签

实例

带有两个提交按钮的表单（进行验证或不进行验证）：

<form action="demo_form.asp" method="get">
E-mail: <input type="email" name="userid" /><br />
<button type="submit" >提交</button><br />
<button type="submit" formnovalidate="formnovalidate">进行没有验证的提交</button>
</form>

亲自试一试

定义和用法

formnovalidate 属性覆盖表单的 novalidate 属性。

如果使用该属性，则提交表单时按钮不会执行验证过程。

该属性与 type="submit" 配合使用。

HTML 4.01 与 HTML 5 之间的差异

formnovalidate 属性是 HTML 5 中的新属性。

语法

<button formnovalidate="formnovalidate">
