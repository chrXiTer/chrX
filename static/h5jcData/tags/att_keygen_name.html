HTML <keygen> name 属性

HTML <keygen> 标签

实例

带有 keygen 字段的表单：

<form action="demo_keygen.asp" method="get">
Username: <input type="text" name="usr_name" />
Encryption: <keygen name="security" />
<input type="submit" />
</form>

亲自试一试

定义和用法

name 属性规定 keygen 元素的名称。

keygen 元素的 name 属性用于在表单数据已被提交后对数据进行引用。

HTML 4.01 与 HTML 5 之间的差异

name 属性是 <keygen> 标签在 HTML5 中的新属性。

语法

<keygen name="value">

属性值

值 描述
name keygen 元素的名称。
