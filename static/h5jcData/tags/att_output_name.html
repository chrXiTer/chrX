定义和用法

name 属性规定 <output> 元素的名称。

name 属性用于在提交表单后引用表单数据，或者在 JavaScript 中引用元素。

HTML 4.01 与 HTML 5 之间的差异

<output> 标签是 HTML5 中的新标签。

语法

<output name="name">

属性值

值 描述
name 规定 <output> 元素的名称。
