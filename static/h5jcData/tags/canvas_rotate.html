定义和用法

rotate() 方法旋转当前的绘图。

JavaScript 语法：

context.rotate(angle);

参数值

参数 描述
angle

旋转角度，以弧度计。

如需将角度转换为弧度，请使用 degrees*Math.PI/180 公式进行计算。

举例：如需旋转 5 度，可规定下面的公式：5*Math.PI/180。
