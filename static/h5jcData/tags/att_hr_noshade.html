HTML <hr> 标签的 noshade 属性

HTML <hr> 标签

实例

没有阴影效果的水平线：

<hr noshade="noshade" />

亲自试一试

定义和用法

noshade 属性规定水平线的颜色呈现为纯色，而不是有阴影的颜色。

兼容性注释

在 HTML 4.01 中，不赞成使用 hr 元素的 noshade 属性；在 XHTML 1.0 Strict DTD 中，不支持 hr 元素的 noshade 属性。

请使用 CSS 代替。

CSS 语法（Internet Explorer）：<hr style="height:2px;color:gray" />

CSS 语法（Firefox、Opera、Chrome 以及 Safari）：<hr style="height:2px;border-width:0;background-color:gray" />

CSS 语法（跨浏览器）：<hr style="height:2px;border-width:0;color:gray;background-color:gray" />

CSS 实例：对齐 hr 元素

在我们的 CSS 教程中，您可以找到更多有关 color 属性 和 background-color 属性 的细节。

语法

<hr noshade="value">

属性值

值 描述
noshade 规定水平线的颜色呈现为纯色（没有阴影）。
