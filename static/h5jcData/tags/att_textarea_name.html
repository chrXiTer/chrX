HTML <textarea> 标签的 name 属性

HTML <textarea> 标签

实例

一个带有 name 属性的文本区：

<textarea rows="3" cols="20" name="W3School_text">
在w3school，你可以找到你所需要的所有的网站建设教程。
</textarea>

亲自试一试

定义和用法

name 属性为文本区规定名称。

name 属性用于在 JavaScript 中对元素进行引用，或者在表单提交之后，对表单数据进行引用。

语法

<textarea name="value">

属性值

值 描述
name_of_textarea 规定文本区的名称。
