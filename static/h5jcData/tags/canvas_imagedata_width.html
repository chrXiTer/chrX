HTML 5 canvas width 属性

HTML5 Canvas 参考手册

实例

输出 ImageData 对象的宽度：

alert("Width of imgData is: " + imgData.width);

亲自试一试

定义和用法

width 属性返回 ImageData 对象的宽度，以像素计。

提示：请参阅 createImageData()、getImageData() 以及 putImageData() 方法，以获得更多关于 ImageData 对象的知识。

默认值： #000000
JavaScript 语法： imgData.width;
