HTML <ol> 标签的 start 属性

HTML <ol> 标签

实例

在有序列表中使用 start 属性：

<ol start="5">
   <li>HTML</li>
   <li>XHTML</li>
   <li>CSS</li>
</ol>

亲自试一试

定义和用法

start 属性规定有序列表的开始点。

兼容性注释

在 HTML 4.01 中，不赞成使用 ol 元素的 start 属性；在 XHTML 1.0 Strict DTD 中，不支持 ol 元素的 start 属性。

注释：目前，仍然没有 start 属性的 CSS 替代方案。

语法

<ol start="value">

属性值

值 描述
number 数字。有序列表的开始点。
