HTML <command> icon 属性

HTML <command> 标签

实例

带有图标图像的 command 元素：

<menu>
  <command onclick="alert('Hello World')" icon="bulbon.gif">
  Click Me!
  </command>
</menu>

亲自试一试

定义和用法

icon 属性规定用于代表 command 元素的图像。

HTML 4.01 与 HTML 5 之间的差异

<command> 标签是 HTML5 中的新标签。

语法

<command icon="URL">

属性值

值 描述
URL

规定用于代表命令的图像的 URL。

可能的值：

  • 绝对 URL - 指向另一个网站（比如 icon="http://www.example.com/lamp.jpg"）
  • 相对 URL - 指向网站内的文件（比如 icon="lamp.jpg"）
