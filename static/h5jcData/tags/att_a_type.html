HTML <a> type 属性

HTML <a> 标签

实例

type 属性规定链接中指向的文档的 mime 类型：

<a href="http://h5jc.sinaapp.com/" type="text/html">W3School</a>

亲自试一试

定义和用法

type 属性规定目标文档的 MIME 类型。

只能在 href 属性存在时使用。

HTML 4.01 与 HTML 5 之间的差异

该属性是 HTML5 中 a 元素的新属性。

语法

<a type="value">

属性值

值 描述
MIME_type

被链接文档的 MIME 类型。

请参阅 IANA MIME types，其中列出了标准的 MIME 类型。
