HTML <th> 标签的 colspan 属性

HTML <th> 标签

实例

表头单元横跨两列的表格：

<table border="1">
  <tr>
    <th colspan="2">Company in USA</th>
  </tr>
  <tr>
    <td>Apple, Inc.</td>
    <td>1 Infinite Loop Cupertino, CA 95014</td>
  </tr>
  <tr>
    <td>Google, Inc.</td>
    <td>1600 Amphitheatre Parkway Mountain View, CA 94043</td>
  </tr>
</table>

亲自试一试

定义和用法

colspan 属性规定表头单元格可横跨的列数。

语法

<th colspan="value">

属性值

值 描述
number

设置单元格可横跨的列数。

注释：colspan="0" 指示浏览器横跨到列组的最后一列。
