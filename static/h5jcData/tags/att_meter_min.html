HTML <meter> min 属性

HTML <meter> 标签

实例

带有当前值以及 min、max、high、low 属性的测量范围：

<meter min="0" low="40" high="90" max="100" value="95"></meter>

亲自试一试

定义和用法

min 属性规定度量的下限。

min 属性值必须小于 max 属性值。

如果未规定，则默认值是 0。

提示：min 属性，与 max 属性一起，规定度量的完整范围。

HTML 4.01 与 HTML 5 之间的差异

<meter> 标签是 HTML 5 中的新标签。

语法

<meter min="number">
值 描述
number 规定度量的最小值，浮点值。默认值是 "0"。
