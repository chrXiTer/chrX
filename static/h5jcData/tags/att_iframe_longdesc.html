HTML <iframe> 标签的 longdesc 属性

HTML <iframe> 标签

实例

带有 longdesc 属性的 iframe，这个属性指向了带有长描述的页面：

<iframe src ="/index.html" longdesc="w3school.txt">
<p>Your browser does not support iframes.</p>
</iframe>

亲自试一试

定义和用法

longdesc 属性规定指向某个页面的 URL，该页面包含了有关 iframe 的较长描述。

由于浏览器对 longdesc 属性的支持性非常差，没有必要使用该属性。

提示：如需为某个 iframe 提供长描述（如果有必要的话），只要简单地创建一个指向描述页面的链接即可（该链接对任何人都是可见的）。

语法

<iframe longdesc="value">

属性值

值 描述
URL

规定 iframe 的内容描述页面的 URL。

可能的值：

  • 绝对 URL - 指向其他站点（比如 src="www.example.com/description.txt"）
  • 相对 URL - 指向站点内的文件（比如 src="description.txt"）
