HTML <base> 标签的 href 属性

HTML <base> 标签

实例

为页面上所有相对 URL 规定基准 URL：

<head>
<base href="/chrX/nofile/123i/" />
</head>

<body>
<img src="eg_smile.gif" />
</body>

亲自试一试

定义和用法

href 属性规定页面中所有相对链接的基准 URL。

语法

<base href="value">

属性值

值 描述
URL 作为基准 URL 的绝对 URL（比如 "http://www.example.com/"）。
