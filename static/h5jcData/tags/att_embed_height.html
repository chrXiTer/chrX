HTML 5 <embed> height 属性

HTML <embed> 标签

实例

高度和宽度设置为 200 像素的 flash 动画：

<embed src="helloworld.swf" width="200" height="200" />

亲自试一试

定义和用法

height 属性规定被嵌入内容的高度。

HTML 4.01 与 HTML 5 之间的差异

<embed> 标签是 HTML5 中的新标签。

语法

<embed height="pixels" />

属性值

值 描述
pixels 规定被嵌入内容的高度，以像素计（比如 "100px" 或 just "100"）
