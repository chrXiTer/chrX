HTML <strike> 标签

定义和用法

<strike> 标签可定义加删除线文本定义。

HTML 与 XHTML 之间的差异

在 HTML 4.01 中，<strike> 标签不被赞成使用。

在 XHTML 1.0 Strict DTD 中，<strike> 标签不被支持。
