HTML <option> 标签的 disabled 属性

HTML <option> 标签

实例

带有 4 个选项的下拉列表，一个选项已被禁用：

<select>
  <option>Volvo</option>
  <option>Saab</option>
  <option disabled="disabled">Opel</option>
  <option>Audi</option>
</select>

亲自试一试

定义和用法

disabled 属性规定某个选项应该被禁用。被禁用的选项既不可用，也不可点击。

可以设置 disabled 属性，直到满足某些条件（比如选择一个复选框），才恢复用户对该选项的使用。然后，可以使用 JavaScript 来清除 disabled 属性，以使选项变为可用状态。

语法

<option disabled="value">

属性值

值 描述
disabled 规定应该禁用选项。
