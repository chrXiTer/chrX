定义和用法

<ul> 标签定义无序列表。

HTML 与 XHTML 之间的差异

在 HTML 4.01 中，ul 元素的 "compact" 和 "type" 属性是不被赞成使用的。

在 XHTML 1.0 Strict DTD 中，ul 元素的 "compact" 和 "type" 属性是不被支持的。

提示和注释：

提示：请使用样式来定义列表的类型。

可选的属性

属性 值 描述
compact compact

不赞成使用。请使用样式取代它。

规定列表呈现的效果比正常情况更小巧。
type
  • disc
  • square
  • circle

不赞成使用。请使用样式取代它。

规定列表的项目符号的类型。
