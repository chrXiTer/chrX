HTML <option> 标签的 selected 属性

HTML <option> 标签

实例

带有预先选定的选项的下拉列表：

<select>
  <option>Volvo</option>
  <option selected="selected">Saab</option>
  <option>Mercedes</option>
  <option>Audi</option>
</select>

亲自试一试

定义和用法

selected 属性规定在页面加载时预先选定该选项。

被预选的选项会显示在下拉列表最前面的位置。

也可以在页面加载后通过 JavaScript 设置 selected 属性。

语法

<option selected="value">

属性值

值 描述
selected 预先选择一个选项。
