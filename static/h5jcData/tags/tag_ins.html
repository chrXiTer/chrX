HTML <ins> 标签

实例

带有已删除部分和新插入部分的文本：

a dozen is <del>20</del> <ins>12</ins> pieces

亲自试一试

定义和用法

<ins> 标签定义已经被插入文档中的文本。

HTML 与 XHTML 之间的差异

NONE

提示和注释：

提示：请与 <del> 一同使用，来描述文档中的更新和修正。

可选的属性

属性 值 描述
cite URL 指向另外一个文档的 URL，此文档可解释文本被插入的原因。
datetime YYYYMMDD 定义文本被插入的日期和时间。

全局属性

<ins> 标签支持 HTML 中的全局属性。

事件属性

<ins> 标签支持 HTML 中的事件属性。
