HTML <command> label 属性

HTML <command> 标签

实例

带有 label 的 command 元素：

<menu>
<command onclick="alert('Hello World')" label="Alert">Click Me!</command>
</menu>

亲自试一试

定义和用法

label 属性规定 command 元素的可见标签（label）。

HTML 4.01 与 HTML 5 之间的差异

<command> 标签是 HTML5 中的新标签。

语法

<command label="text">

属性值

值 描述
checked 规定命令的可见标签。
