HTML 5 Audio/Video DOM preload 属性

HTML 5 Audio/Video DOM 参考手册

实例

页面一旦加载，就开始加载视频：

myVid=document.getElementById("video1");
myVid.preload="auto";

亲自试一试

定义和用法

preload 属性设置或返回是否在页面加载后立即加载音频/视频。

语法

设置 preload 属性：

audio|video.preload="auto|metadata|none"

返回 preload 属性：

audio|video.preload

属性值

值 描述
auto 指示一旦页面加载，则开始加载音频/视频。
metadata 指示当页面加载后仅加载音频/视频的元数据。
none 指示页面加载后不应加载音频/视频。

返回值

类型 描述
字符串值 auto|metadata|none
