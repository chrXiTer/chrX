HTML <keygen> form 属性

HTML <keygen> 标签

实例

下面这个 keygen 字段属于 id 为 "secureform" 的表单：

Encryption: <keygen name="security" form="secureform" />

亲自试一试

定义和用法

form 属性规定 keygen 字段所属的一个或多个表单。

form 属性的值必须是其所属的表单的 id。

如需引用一个以上的表单，请使用空格分隔的列表。

HTML 4.01 与 HTML 5 之间的差异

form 属性是 <keygen> 标签在 HTML5 中的新属性。

语法

<keygen form="value">

属性值

值 描述
id keygen 字段所属的一个或多个表单的 id。
