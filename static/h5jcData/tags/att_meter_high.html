HTML <meter> high 属性

HTML <meter> 标签

实例

带有当前值以及 min、max、high、low 属性的测量范围：

<meter min="0" low="40" high="90" max="100" value="95"></meter>

亲自试一试

定义和用法

high 属性规定的范围是被视为高值的度量值。

high 属性必须小于 max 属性值，且必须小于 low 和 min 属性值。

HTML 4.01 与 HTML 5 之间的差异

<meter> 标签是 HTML 5 中的新标签。

语法

<meter high="number">
值 描述
number 规定被视作高值的浮点数。
