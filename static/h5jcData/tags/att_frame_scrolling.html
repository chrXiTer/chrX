HTML <frame> 标签的 scrolling 属性

HTML <frame> 标签

实例

总是显示滚动条的框架：

<html>

<frameset cols="50%,50%">
<frame src="frame_a.htm" scrolling="yes" />
<frame src="frame_b.htm" />
</frameset>

</html>

亲自试一试

定义和用法

scrolling 属性规定是否在框架中显示滚动条。

默认地，如果内容大于框架，就会出现滚动条。

语法

<frame scrolling="value">

属性值

值 描述
auto 在需要的时候显示滚动条。
yes 始终显示滚动条（即使不需要）。
no 从不显示滚动条（即使需要）。
