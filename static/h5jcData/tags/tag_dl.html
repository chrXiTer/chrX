定义和用法

<dl> 标签定义了定义列表（definition list）。

<dl> 标签用于结合 <dt> （定义列表中的项目）和 <dd> （描述列表中的项目）。

HTML 与 XHTML 之间的差异

NONE

全局属性

<dl> 标签支持 HTML 中的全局属性。

事件属性

<dl> 标签支持 HTML 中的事件属性。
