HTML 5 Audio/Video DOM pause() 方法

HTML 5 Audio/Video DOM 参考手册

实例

带有播放和暂停按钮的一段视频：

var myVideo=document.getElementById("video1");
function playVid()
  {
  myVideo.play();
  }
function pauseVid()
  {
  myVideo.pause();
  }

亲自试一试

定义和用法

pause() 方法停止（暂停）当前播放的音频或视频。

语法

audio|video.pause()
