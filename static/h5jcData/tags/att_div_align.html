HTML <div> 标签的 align 属性

HTML <div> 标签

实例

文档中的一个部分居中对齐：

<div align="center">
  This is some text!
</div>

亲自试一试

定义和用法

align 属性规定 div 元素中的内容的水平对齐方式。

兼容性注释

在 HTML 4.01 中，不赞成使用 div 元素的 align 属性；在 XHTML 1.0 Strict DTD 中，不支持 div 元素的 align 属性。

请使用 CSS 代替。

CSS 语法：<div style="text-align:center">

CSS 实例：对齐 div 元素中的文本

在我们的 CSS 教程中，您可以找到更多有关 text-align 属性的细节。

语法

<div align="value">

属性值

值 描述
left 左对齐内容。
right 右对齐内容。
center 居中对齐内容。
justify 对行进行伸展，这样每行都可以有相等的长度（就像在报纸和杂志中）。
