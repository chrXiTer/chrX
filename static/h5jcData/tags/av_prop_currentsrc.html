HTML 5 Audio/Video DOM currentSrc 属性

HTML 5 Audio/Video DOM 参考手册

实例

获得当前视频的 URL：

myVid=document.getElementById("video1");
alert(myVid.currentSrc);

亲自试一试

定义和用法

currentSrc 熟悉返回当前音频/视频的 URL。

如果未设置音频/视频，则返回空字符串。

语法

audio|video.currentSrc

技术细节

返回值 字符串，表示音频/视频的当前 URL
