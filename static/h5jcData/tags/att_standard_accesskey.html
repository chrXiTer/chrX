定义和用法

accesskey 属性规定激活（使元素获得焦点）元素的快捷键。

提示和注释

注释：以下元素支持 accesskey 属性：<a>, <area>, <button>, <input>, <label>, <legend> 以及 <textarea>。

语法

<element accesskey="character">

属性值

值 描述
character 规定激活（使元素获得焦点）元素的便捷按键。
