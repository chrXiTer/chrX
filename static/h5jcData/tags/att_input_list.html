定义和用法

list 属性引用数据列表，其中包含输入字段的预定义选项。

HTML 4.01 与 HTML 5 之间的差异

list 属性是 <input> 标签在 HTML5 中的新属性。

语法

<input list="value">

属性值

值 描述
datalist-id 文档中的 datalist 的 id。
