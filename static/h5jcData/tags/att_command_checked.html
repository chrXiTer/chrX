HTML <command> checked 属性

HTML <command> 标签

实例

一个预先选择复选框的 command 元素：

<menu>
  <command onclick="alert('Hello World')" type="checkbox" checked="checked">
  Click Me!
  </command>
</menu>

亲自试一试

定义和用法

checked 属性规定应该预先选择 command 元素。

只有当命令类型设置为 checkbox 或 radio 时，才能使用该属性。

HTML 4.01 与 HTML 5 之间的差异

<command> 标签是 HTML5 中的新标签。

语法

<command checked="checked">

属性值

值 描述
checked 规定应该预先选择该命令（适用于 checkbox 或 radio）
