HTML <keygen> autofocus 属性

HTML <keygen> 标签

实例

当页面加载时获得焦点的 keygen 字段：

<form action="demo_keygen.asp" method="get">
Username: <input type="text" name="usr_name" />
Encryption: <keygen name="security" autofocus="autofocus" />
<input type="submit" />
</form>

亲自试一试

定义和用法

autofocus 属性规定当页面加载时，keygen 字段会自动获得焦点。

如果使用该属性，则字段会获得焦点。

HTML 4.01 与 HTML 5 之间的差异

autofocus 属性是 <keygen> 标签在 HTML5 中的新属性。

语法

<keygen autofocus="autofocus">

HTML <keygen> 标签
