HTML <style> 标签的 type 属性

HTML <style> 标签

实例

在 <style> 标签中使用 type 属性：

<html>
<head>
<style type="text/css">
h1 {color:red}
p {color:blue}
</style>
</head>

<body>
<h1>Header 1</h1>
<p>A paragraph.</p>
</body>
</html>

定义和用法

必需的 type 属性规定样式表的 MIME 类型。

type 属性指示 <style> 与 </style> 标签之间的内容。

值 "text/css" 指示内容是标准的 CSS。

语法

<style type="value">

属性值

值 描述
MIME_type 样式表的 MIME 类型。目前，唯一可能的值是 "text/css"。
