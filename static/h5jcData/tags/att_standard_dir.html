HTML dir 属性

HTML 全局属性

实例

一段方向从右向左的段落：

<p dir="rtl">Write this text right-to-left!</p>

亲自试一试

定义和用法

dir 属性规定元素内容的文本方向。

提示和注释

注释：dir 属性在以下标签中无效：<base>, <br>, <frame>, <frameset>, <hr>, <iframe>, <param> 以及 <script>。

语法

<element dir="ltr|rtl">

属性值

值 描述
ltr 默认。从左向右的文本方向。
rtl 从右向左的文本方向。
