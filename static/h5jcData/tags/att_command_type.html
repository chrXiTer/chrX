HTML <command> type 属性

HTML <command> 标签

实例

一个复选框 command 元素：

<menu>
<command onclick="alert('Hello World')" type="checkbox">Click Me!</command>
</menu>

亲自试一试

定义和用法

type 属性规定 command 的类型。

type 属性指示命令的类型：既可以是带有相关动作的普通命令，也可以是能够被切换的状态或选项，或者是对列表中一个项目的选择。

该属性是带有三个关键词和状态的枚举属性。"command" 关键词映射 Command 状态，"checkbox" 关键词映射 Checkbox 状态，而 "radio" 关键词映射 Radio 状态。缺省的默认值是 Command 状态。

Command 状态
元素表示带有相关联动作的普通命令。
Checkbox 状态
元素表示可被切换的状态或选项。
Radio 状态
元素表示对列表项中一个项目的选择。

HTML 4.01 与 HTML 5 之间的差异

<command> 标签是 HTML5 中的新标签。

语法

<command type="command|checkbox|radio">

属性值

值 描述
command 默认。规定带有 action 的普通 command。
checkbox 规定可切换的选项。
radio 规定列表中的单项的选择。
