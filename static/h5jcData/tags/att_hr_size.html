HTML <hr> 标签的 size 属性

HTML <hr> 标签

实例

高度为 50 像素的水平线：

<hr size="50" />

亲自试一试

定义和用法

size 属性规定水平线的高度，以像素计。

兼容性注释

在 HTML 4.01 中，不赞成使用 hr 元素的 size 属性；在 XHTML 1.0 Strict DTD 中，不支持 hr 元素的 size 属性。

请使用 CSS 代替。

CSS 语法：<hr style="height:30px" />

CSS 实例：设置 hr 元素的高度

在我们的 CSS 教程中，您可以找到更多有关 height 属性 的细节。

语法

<hr size="value">

属性值

值 描述
pixels 规定 hr 元素的高度。
