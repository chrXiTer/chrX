HTML <ul> 标签的 compact 属性

HTML <ul> 标签

实例

在无序列表中使用 compact 属性：

<ul compact="compact">
   <li>HTML</li>
   <li>XHTML</li>
   <li>CSS</li>
</ul>

亲自试一试

定义和用法

compact 属性规定列表呈现的效果比正常情况更小巧，这是通过减少行间距以及对列表进行缩进来实现的。

兼容性注释

在 HTML 4.01 中，不赞成使用 ul 元素的 compact 属性；在 XHTML 1.0 Strict DTD 中，不支持 ul 元素的 compact 属性。

请使用 CSS 代替。

CSS 语法：<ul style="line-height: 80%">

CSS 实例：缩小列表中的行高

在我们的 CSS 教程中，您可以找到更多有关 line-height 属性 的细节。

语法

<ul compact="value">

属性值

值 描述
compact 规定列表呈现更紧凑小巧的效果。
