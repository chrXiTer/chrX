定义和用法

strokeRect() 方法绘制矩形（不填色）。笔触的默认颜色是黑色。

提示：请使用 strokeStyle 属性来设置笔触的颜色、渐变或模式。

JavaScript 语法：

context.strokeRect(x,y,width,height);

参数值

参数 描述
x 矩形左上角的 x 坐标
y 矩形左上角的 y 坐标
width 矩形的宽度，以像素计
height 矩形的高度，以像素计
