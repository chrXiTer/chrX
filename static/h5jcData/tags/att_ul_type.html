HTML <ul> 标签的 type 属性

HTML <ul> 标签

实例

在无序列表中使用 type 属性：

<ul type="square">
   <li>HTML</li>
   <li>XHTML</li>
   <li>CSS</li>
</ul>

亲自试一试

定义和用法

type 属性规定列表的项目符号的类型。

兼容性注释

在 HTML 4.01 中，不赞成使用 ul 元素的 type 属性；在 XHTML 1.0 Strict DTD 中，不支持 ul 元素的 type 属性。

请使用 CSS 代替。

CSS 语法：<ul style="list-style-type:square">

CSS 实例：列表样式类型

在我们的 CSS 教程中，您可以找到更多有关 list-style-type 属性 的细节。

语法

<ul type="value">

属性值

值 描述
disc 默认值。实心圆。
circle 空心圆。
square 实心方块。
