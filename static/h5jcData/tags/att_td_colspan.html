HTML <td> 标签的 colspan 属性

HTML <td> 标签

实例

表格单元横跨两列的表格：

<table border="1">
  <tr>
    <th>Month</th>
    <th>Savings</th>
  </tr>
  <tr>
    <td colspan="2">January</td>
  </tr>
  <tr>
    <td colspan="2">February</td>
  </tr>
</table>

亲自试一试

定义和用法

colspan 属性规定单元格可横跨的列数。

语法

<td colspan="value">

属性值

值 描述
number

设置单元格可横跨的列数。

注释：colspan="0" 指示浏览器横跨到列组的最后一列。
