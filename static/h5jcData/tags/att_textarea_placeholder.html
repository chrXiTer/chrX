HTML <textarea> 标签的 placeholder 属性

HTML <textarea> 标签

实例

带有占位符文本的文本区域：

<textarea placeholder="请介绍自己...">
</textarea>

亲自试一试

定义和用法

placeholder 属性规定描述文本区域预期值的简短提示。

该提示会在文本区域为空时显示，当字段获得焦点时消失。

HTML 4.01 与 HTML 5 之间的差异

placeholder 属性是 <textarea> 标签在 HTML5 中的新属性。

语法

<textarea placeholder="text">

属性值

值 描述
text 规定描述文本区域预期值的简短提示。
