HTML onblur 事件属性

HTML 事件属性

实例

当用户离开输入字段时对其进行验证：

<input type="text" name="fname" id="fname" onblur="upperCase()">

亲自试一试

定义和用法

onblur 属性在元素失去焦点时触发。

onblur 常用于表单验证代码（例如用户离开表单字段）。

提示：onblur 属性与 onfocus 属性相反。

HTML 4.01 与 HTML5 之间的差异

无。

语法

<element onblur="script">

属性值

值 描述
script onblur 发生时运行的脚本。
