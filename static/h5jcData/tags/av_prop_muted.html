HTML 5 Audio/Video DOM muted 属性

HTML 5 Audio/Video DOM 参考手册

实例

关闭视频的声音：

myVid=document.getElementById("video1");
myVid.muted=true;

亲自试一试

定义和用法

muted 属性设置或返回音频/视频是否应该被静音（关闭声音）。

语法

设置 muted 属性：

audio|video.muted=true|false

返回 muted 属性：

audio|video.muted

属性值

值 描述
true 指示应该关闭音频/视频的声音。
false 默认。指示应该打开音频/视频的声音。

技术细节

返回值 布尔值。true|false
默认值： false
