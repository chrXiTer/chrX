HTML 5 <button> formmethod 属性

HTML <button> 标签

实例

覆盖表单 HTTP 方法的按钮：

<form action="demo_form.asp" method="get">
  First name: <input type="text" name="fname" />
  Last name: <input type="text" name="lname" />
<button type="submit" >提交</button>
<button type="submit" formmethod="post" formaction="demo_post.asp">
使用 POST 来提交</button>
</form>

亲自试一试

定义和用法

formmethod 属性覆盖 form 元素的 method 属性。

可以通过以下方式发送 form-data ：

  • 以 URL 变量 (使用 method="get") 的形式来发送
  • 以 HTTP post (使用 method="post") 的形式来发送

该属性与 type="submit" 配合使用。

HTML 4.01 与 HTML 5 之间的差异

formmethod 属性是 HTML 5 中的新属性。

语法

<button formmethod="get|post">

属性值

值 描述
get 向 URL 追加表单数据（form-data）：URL?name=value&name=value
post 以 HTTP post 事务的形式发送表单数据（form-data）
