HTML <command> radiogroup 属性

HTML <command> 标签

实例

command 元素组：

<menu>
  <command onclick="alert('Hello World')" type="radio" radiogroup="group1">
    Click Me!
  </command>
  <br />
  <command onclick="alert('Good morning!')">
    Click This!
  </command>
  <br />
  <command onclick="alert('Good day!')" type="radio" radiogroup="group1">
    And this!
  </command>
</menu>

亲自试一试

定义和用法

radiogroup 属性规定命令组的名称，在该命令被激活或切换时，该命令组会被切换。

该属性只能用于类型为 "radio" 的普通命令。

HTML 4.01 与 HTML 5 之间的差异

<command> 标签是 HTML5 中的新标签。

语法

<command radiogroup="groupname">

属性值

值 描述
groupname 规定将被切换的命令组的名称（仅适用于 type="radio"）。
