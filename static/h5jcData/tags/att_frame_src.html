HTML <frame> 标签的 src 属性

HTML <frame> 标签

实例

src 属性规定在框架中显示的文档的位置：

<html>

<frameset cols="50%,50%">
<frame src="frame_a.htm" />
<frame src="frame_b.htm" />
</frameset>

</html>

亲自试一试

定义和用法

src 属性规定在框架中显示的文档的 URL。

语法

<frame src="value">

属性值

值 描述
URL

规定要在框架中显示的文档的地址。

可能的值：

  • 绝对 URL - 指向其他站点（比如 src="www.example.com/index.html"）
  • 相对 URL - 指向站点内的文件（比如 src="index.html"）
