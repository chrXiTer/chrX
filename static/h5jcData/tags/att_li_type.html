HTML <li> 标签的 type 属性

HTML <ul> 标签

实例

在无序和有序的 HTML 列表中使用 type 属性：

<ol>
  <li>Coffee</li>
  <li type="a">Tea</li>
  <li>Milk</li>
</ol>

<ul>
  <li>Coffee</li>
  <li type="square">Tea</li>
  <li>Milk</li>
</ul>

亲自试一试

定义和用法

type 属性规定列表中的列表项目的项目符号的类型。

兼容性注释

在 HTML 4.01 中，不赞成使用 li 元素的 type 属性；在 XHTML 1.0 Strict DTD 中，不支持 li 元素的 type 属性。

请使用 CSS 代替。

CSS 语法：<li style="list-style-type:square">

CSS 实例：列表样式类型

在我们的 CSS 教程中，您可以找到更多有关 list-style-type 属性 的细节。

语法

<li type="value">

属性值

有序列表的属性值：

值 描述
1 数字顺序的有序列表（默认值）（1, 2, 3, 4）。
a 字母顺序的有序列表，小写（a, b, c, d）。
A 字母顺序的有序列表，大写（a, b, c, d）。
i 罗马数字，小写（i, ii, iii, iv）。
I 罗马数字，大写（i, ii, iii, iv）。

无序列表的属性值：

值 描述
disc 默认值。实心圆。
circle 空心圆。
square 实心方块。
