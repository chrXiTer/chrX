定义和用法

form 属性规定 <object> 元素所属的一个或多个表单。

HTML 4.01 与 HTML 5 之间的差异

form 属性是 HTML5 中的新属性。

语法

<object form="form_id">

属性值

值 描述
form_id 规定 <object> 元素所属的一个或多个表单的 id 列表，以空格分隔。
