HTML <ol> 标签的 type 属性

HTML <ol> 标签

实例

在有序列表中使用 type 属性：

<ol type="I">
   <li>HTML</li>
   <li>XHTML</li>
   <li>CSS</li>
</ol>

亲自试一试

定义和用法

type 属性规定有序列表的项目符号的类型。

兼容性注释

在 HTML 4.01 中，不赞成使用 ol 元素的 type 属性；在 XHTML 1.0 Strict DTD 中，不支持 ol 元素的 type 属性。

请使用 CSS 代替。

CSS 语法：<ol style="list-style-type:upper-roman">

CSS 实例：列表样式类型

在我们的 CSS 教程中，您可以找到更多有关 list-style-type 属性 的细节。

语法

<ol type="value">

属性值

值 描述
1 默认值。数字有序列表。（1、2、3、4）
a 按字母顺序排列的有序列表，小写。（a、b、c、d）
A 按字母顺序排列的有序列表，大写。（A、B、C、D）
i 罗马字母，小写。（i, ii, iii, iv）
I 罗马字母，大写。（I, II, III, IV）
