JavaScript log() 方法

JavaScript Math 对象

定义和用法

log() 方法可返回一个数的自然对数。

语法

Math.log(x)
参数 描述
x 必需。任意数值或表达式。

返回值

x 的自然对数。

说明

参数 x 必须大于 0。

实例

在本例中，我们将对不同的数运用 log() ：

<script type="text/javascript">

document.write(Math.log(2.7183) + "<br />")
document.write(Math.log(2) + "<br />")
document.write(Math.log(1) + "<br />")
document.write(Math.log(0) + "<br />")
document.write(Math.log(-1))

</script>

输出：

1.0000066849139877
0.6931471805599453
0
-Infinity
NaN
