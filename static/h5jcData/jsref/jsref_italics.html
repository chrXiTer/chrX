JavaScript italics() 方法

JavaScript String 对象

定义和用法

italics() 方法用于把字符串显示为斜体。

语法

stringObject.italics()

实例

在本例中，"Hello world!" 将被显示为斜体：

<script type="text/javascript">

var str="Hello world!"
document.write(str.italics())

</script>
