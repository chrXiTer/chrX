JavaScript 全局对象

全局对象描述

全局对象是预定义的对象，作为 JavaScript 的全局函数和全局属性的占位符。在 JavaScript 代码嵌入一个特殊环境中时，全局对象通常具有环境特定的属性，甚至不同的名字，在浏览器中这个全局变量就是window，在其它环境中可能是其他名字。

在最顶层的 JavaScript 代码中，还可以用关键字 this 引用全局对象。但平常使用全局对象中的属性（全局属性）和方法（全局方法），并不需要直接引用全局对象。因为全局对象是作用域链的头，所有非限定性的变量和函数名都会作为该对象的属性来查询。例如，当JavaScript 代码引用 parseInt() 函数时，它引用的是全局对象的 parseInt 属性。全局对象是作用域链的头，还意味着在顶层 JavaScript 代码中声明的所有变量都将成为全局对象的属性。

在 JavaScript 核心语言中，全局对象的预定义属性都是不可枚举的，但可以用 for/in 循环列出所有隐式或显式声明的全局变量，如下所示：

var variables = "";
for (var name in this) {
	variables += name + "<br />";
}
document.write(variables);

顶层函数（全局函数）

函数 描述
decodeURI() 解码某个编码的 URI。
decodeURIComponent() 解码一个编码的 URI 组件。
encodeURI() 把字符串编码为 URI。
encodeURIComponent() 把字符串编码为 URI 组件。
escape() 对字符串进行编码。
eval() 计算 JavaScript 字符串，并把它作为脚本代码来执行。
getClass() 返回一个 JavaObject 的 JavaClass。
isFinite() 检查某个值是否为有穷大的数。
isNaN() 检查某个值是否是数字。
Number() 把对象的值转换为数字。
parseFloat() 解析一个字符串并返回一个浮点数。
parseInt() 解析一个字符串并返回一个整数。
String() 把对象的值转换为字符串。
unescape() 对由 escape() 编码的字符串进行解码。

方法 描述
