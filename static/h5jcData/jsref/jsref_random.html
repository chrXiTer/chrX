JavaScript random() 方法

JavaScript Math 对象

定义和用法

random() 方法可返回介于 0 ~ 1 之间的一个随机数。

语法

Math.random()

返回值

0.0 ~ 1.0 之间的一个伪随机数。

实例

在本例中，我们将取得介于 0 到 1 之间的一个随机数：

<script type="text/javascript">

document.write(Math.random())

</script>
