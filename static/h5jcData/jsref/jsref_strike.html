JavaScript strike() 方法

JavaScript String 对象

定义和用法

strike() 方法用于显示加删除线的字符串。

语法

stringObject.strike()

实例

在本例中，"Hello world!" 将被加上一条删除线：

<script type="text/javascript">

var str="Hello world!"
document.write(str.strike())

</script>
