JavaScript RegExp \xxx 元字符

JavaScript RegExp 对象

定义和用法

\xxx 元字符用于查找以八进制数 xxx 规定的字符。

如果未找到匹配，则返回 null。

语法

new RegExp("\xxx")

直接量语法：

/\xxx/

浏览器支持

所有主流浏览器都支持 \xxx 元字符。

实例

对字符串中的八进制 127 (W) 进行全局搜索：

var str="Visit W3School. Hello World!";
var patt1=/\127/g;

下面被标记的文本显示了表达式获得匹配的位置：

Visit W3School. Hello World!
