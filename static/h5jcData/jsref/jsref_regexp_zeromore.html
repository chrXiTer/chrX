JavaScript RegExp * 量词

JavaScript RegExp 对象

定义和用法

n+ 量词匹配包含零个或多个 n 的任何字符串。

语法

new RegExp("n*")

直接量语法：

/n*/

浏览器支持

所有主流浏览器都支持 * 量词。

实例

例子 1

对 "l" 进行全局搜索，包括其后紧跟的一个或多个 "o"：

var str="Hellooo World! Hello W3School!";
var patt1=/lo*/g;

下面被标记的文本显示了表达式获得匹配的位置：

Hellooo World! Hello W3School!
