HTML DOM print() 方法

HTML DOM Window 对象

定义和用法

print() 方法用于打印当前窗口的内容。

调用 print() 方法所引发的行为就像用户单击浏览器的打印按钮。通常，这会产生一个对话框，让用户可以取消或定制打印请求。

语法

window.print()

实例 1

<html>
<head>
<script type="text/javascript">
function printpage()
  {
  window.print()
  }
</script>
</head>
<body>

<input type="button" value="Print this page"
onclick="printpage()" />

</body>
</html>
