JavaScript blink() 方法

JavaScript String 对象

定义和用法

blink() 方法用于显示闪动的字符串。

语法

stringObject.blink()

提示和注释

注释：此方法无法工作于 Internet Explorer 中。

实例

在本例中，"Hello world!" 将被显示为闪动的文本：

<script type="text/javascript">

var str="Hello world!"
document.write(str.blink())

</script>
