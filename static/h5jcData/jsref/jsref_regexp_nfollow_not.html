JavaScript RegExp ?! 量词

JavaScript RegExp 对象

定义和用法

?!n 量词匹配其后没有紧接指定字符串 n 的任何字符串。

语法

new RegExp("regexp(?!n)")

直接量语法：

/regexp(?!n)/

浏览器支持

所有主流浏览器都支持 ?! 量词。

实例

对其后没有紧跟 "all" 的 "is" 进行全局搜索：

var str="Is this all there is";
var patt1=/is(?! all)/gi;
