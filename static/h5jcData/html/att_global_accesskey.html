HTML 5 全局属性

实例

带有指定 accesskey 的链接：

<a href="/chrX/nofile/123" accesskey="w">W3School</a><br />
<a href="http://www.google.com/" accesskey="g">Google</a>

亲自试一试

定义和用法

accesskey 属性规定元素的快捷键。

快捷键可以使元素获得焦点，或者激活元素。

HTML 4.01 与 HTML 5 之间的差异

None.

语法

<element accesskey="value">

属性值

值 描述
character 由空格分隔的一个或多个键盘符。
