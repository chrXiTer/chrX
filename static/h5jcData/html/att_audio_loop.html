定义和用法

loop 属性规定当音频结束后将重新开始播放。

如果设置该属性，则音频将循环播放。

HTML 4.01 与 HTML 5 之间的差异

loop 属性是 HTML 5 中的新属性。

语法

<audio loop="loop" />
