定义数组

可以使用关键词 new 来创建数组对象。下面的代码定义了一个名为 myArray 的数组对象：

var myArray=new Array()

也可以使用［］符号来创建数组对象。下面一行代码的效果与上面一行代码的效果完全相同。

var myArray=［］

创建数组时也可以使用一个整数变量定义数组的初始长度：

var mycars=new Array(3)  //没有对应的［］创建方式

创建数组时还可以直接初始化数组中的元素值：


var mycars=new Array("Saab","Volvo","BMW")


var mycars=［"Saab","Volvo","BMW"］



使用数组


可以用［］包围数字索引获取数据中元素，并能够对这些元素赋值


var mycars=new Array(1)
mycars[0]="Saab"
mycars[2]="BMW"
mycars[1]="Volvo"




访问数组


通过指定数组名以及索引号码，你可以访问某个特定的元素。


下面是代码行：


document.write(mycars[0])


下面是输出：


Saab



修改已有数组中的值


如需修改已有数组中的值，只要向指定下标号添加一个新值即可：


mycars[0]="Opel";


现在，以上代码：


document.write(mycars[0]);


将输出：


Opel
