JavaScript 简介

JavaScript 是世界上最流行的编程语言，是Web前端最重要的编程语言，JavaScript 可插入到 HTML 页面中使用。

以下是在浏览器上使用JavaScript的简要介绍：


JavaScript：写入 HTML 输出

document.write("<h1>This is a heading</h1>");
document.write("<p>This is a paragraph</p>");

试一试

提示：只有在 HTML 输出过程中使用 document.write，才能添加内容。如果在文档加载完成之后使用该方法，会覆盖整个文档。

JavaScript：对事件作出反应

<button type="button"
        onclick="alert('Welcome!')">点击这里
</button>

试一试

onclick 事件在用户点击后调用绑定的代码。上例中调用的alert() 函数，使用方便，但对用户很不友好，除了临时的测试演示之外，在正式的编码中不应该使用。

JavaScript：改变 HTML 内容

x=document.getElementById("demo")  //查找元素
x.innerHTML="Hello JavaScript";    //改变内容

亲自试一试

document.getElementByID("some id")会经常看到。这个方法是 HTML DOM 中定义的，是用以访问 HTML 元素标准。

JavaScript：改变 HTML 图像

动态地改变 HTML <image> 的图片来源 (src)。
点击灯泡来点亮或熄灭这盏灯：

点击事件代码如下：

function changeImage(){
  element=document.getElementById('myimage')
  if(element.src.match("bulbon")){
    element.src="../i/eg_bulboff.gif";
  }else{
    element.src="../i/eg_bulbon.gif";
  }
}

亲自试一试

JavaScript 能够改变任意 HTML 元素的大多数属性，而不仅仅是图片。

JavaScript：改变 HTML 样式

改变 HTML 元素的样式，属于改变 HTML 属性的变种。

x=document.getElementById("demo")  //找到元素
x.style.color="#ff0000";           //改变样式

试一试

JavaScript：验证输入

JavaScript 常用于验证用户的输入。

if(isNaN(x)) {alert("Not Numeric")};
