CSS border-right-width 属性

CSS 参考手册

定义和用法

border-right-width 属性设置元素的右边框的宽度。

只有当边框样式不是 none 时才起作用。如果边框样式是 none，边框宽度实际上会重置为 0。不允许指定负长度值。

注释：请始终在 border-right-width 属性之前声明 border-style 属性。元素只有在获得边框之后，才能改变其边框的宽度。

默认值： medium
继承性： no
版本： CSS1
JavaScript 语法： object.style.borderRightWidth="thick"

实例

设置右边框的宽度：

p
  {
  border-style:solid;
  border-right-width:15px;
  }

TIY

浏览器支持

所有浏览器都支持 border-right-width 属性。

注释：任何版本的 Internet Explorer （包括 IE8）都不支持属性值 "inherit"。

可能的值

值 描述
thin 定义细的右边框。
medium 默认值。定义中等的右边框。
thick 定义粗的右边框。
length 允许您自定义右边框的宽度。
inherit 规定应该从父元素继承边框宽度。
