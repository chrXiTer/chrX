CSS 简介

CSS 概述

CSS 指层叠样式表 (Cascading Style Sheets)把样式添加到 HTML中，并实现内容与表现分离的问题
样式定义如何显示 HTML 元素，样式通常存储在样式表中，外部样式表通常存储在 CSS 文件中

CSS 的最新版本为CSS3。 CSS3 是一个通俗的说法。由于CSS3以后，CSS被划分为多个模块，单独形成标准，从形式上来说，已经不存在 CSS3 标准自身。

CSS3有些模块是在CSS2的基础上,模块标准一般从Level 3开始。有些模块（比如选择器）已经在制定Level 4标准。有些模块是CSS2中没有的，其模块标准一般从Level 1开始（比如多列布局）

使用CSS的好处

HTML 标签原本被设计为用于定义文档内容。通过使用 <h1>、<p>、<table> 这样的标签，HTML 的初衷是表达“这是标题”、“这是段落”、“这是表格”之类的信息。同时文档布局由浏览器来完成。

早期，两种主要的浏览器（Netscape 和 Internet Explorer）不断地将新的 HTML 标签和属性（比如字体标签和颜色属性）添加到 HTML 规范中。在带来丰富显示效果的同时，也破坏了HTML文档的语义特性，创建文档内容清晰地独立于文档表现层的站点变得越来越困难。为了解决这个问题，CSS被引入进来。

样式可以保存在外部的 .css 文件中。并可以用于控制多个页面的显示和布局，通过为HTML 元素定义样式，能够将之应用于多个页面中。如需进行全局的更新，只需简单地改变样式，然后各个网页中的相应元素均会自动地更新。

多个样式层叠为一个

CSS允许以多种方式在多个地方规定HTML的样式。可以在具体的 HTML 元素中、在 HTML 页的头元素中，或者外部的 CSS 文件中，甚至可以在同一个 HTML 文档内部引用多个外部样式表。

层叠次序

一般而言，所有的样式会根据下面的规则层叠于一个新的虚拟样式表中，其中数字 4 拥有最高的优先权。

  1. 浏览器缺省设置
  2. 外部样式表
  3. 内部样式表（位于 <head> 标签内部）
  4. 内联样式（在 HTML 元素内部）

因此，内联样式（在 HTML 元素内部）拥有最高的优先权，这意味着它将优先于以下的样式声明：<head> 标签中的样式声明，外部样式表中的样式声明，或者浏览器中的样式声明（缺省值）。
