浏览器支持

所有主流浏览器都支持 text-overflow 属性。

定义和用法

text-overflow 属性规定当文本溢出包含元素时发生的事情。

默认值： clip
继承性： no
版本： CSS3
JavaScript 语法： object.style.textOverflow="ellipsis"

语法

text-outline: clip|ellipsis|string;
值 描述 测试
clip 修剪文本。 测试
ellipsis 显示省略符号来代表被修剪的文本。 测试
string 使用给定的字符串来代表被修剪的文本。
