CSS 伪类 (Pseudo-classes)、CSS 伪元素 (Pseudo-elements)

伪类和伪元素是选择器的补充，能够以更加灵活的方式匹配普通选择器无法匹配的特色场景和内容。
伪类相比普通选择器，增加了特殊的类型。
普通选择器只能匹配元素，利用伪元素可以匹配元素中更加精细的内容。

语法

    selector:pseudo-class {property: value}
    selector:pseudo-element {property:value;}

锚伪类

链接的不同状态都可以用不同的方式显示，这些状态包括：活动状态，已被访问状态，未被访问状态，和鼠标悬停状态。

    a:link {color: #FF0000}	/* 未访问的链接 */
    a:visited {color: #00FF00}	/* 已访问的链接 */
    a:hover {color: #FF00FF}	/* 鼠标移动到链接上 */
    a:active {color: #0000FF}	/* 选定的链接 */

试一试

锚伪类的顺序

由于鼠标悬停状态一定处在活动状态或未被访问状态，因此a:hover 不能放在 a:link 和 a:visited 之前，否则其样式会被a:link 或 a:visited 的样式覆盖。 同样的，活动状态一定处在鼠标悬停状态，因此a:active 不能放在 a:hover 之前，否则其样式会被a:hover 的样式覆盖。

:first-child 伪类

:first-child 伪类选择这样的元素：该元素是父节点中第一个子元素的元素。

提示： p:first-child 选择器选择的不是p的子元素，而是符合前面要求的p元素。

下面举例说明举例来说明。考虑以下标记：

<div>
<p>These are the necessary steps:</p>
<ul>
<li>Intert Key</li>
<li>Turn key <strong>clockwise</strong></li>
<li>Push accelerator</li>
</ul>
<p>Do <em>not</em> push the brake at the same time as the accelerator.</p>
</div>

给定以下规则：

p:first-child {font-weight: bold;}
li:first-child {text-transform:uppercase;}

第一个规则选择所有 p 元素中是父元素中第一个元素的哪些p元素。第一个规则选择所有 li 元素中是父元素中第一个元素的哪些li元素。试一试

:first-line 伪元素

"first-line" 伪元素用于向文本的首行设置特殊样式。

在下面的例子中，浏览器会根据 "first-line" 伪元素中的样式对 p 元素的第一行文本进行格式化：

p:first-line{
  color:#ff0000;
  font-variant:small-caps;
}

试一试

注释："first-line" 伪元素只能用于块级元素。

注释：下面的属性可应用于 "first-line" 伪元素：

  • font
  • color
  • background
  • word-spacing
  • letter-spacing
  • text-decoration
  • vertical-align
  • text-transform
  • line-height
  • clear

:first-letter 伪元素

"first-letter" 伪元素用于向文本的首字母设置特殊样式：

p:first-letter{
  color:#ff0000;
  font-size:xx-large;
}

试一试

注释："first-letter" 伪元素只能用于块级元素。

注释：下面的属性可应用于 "first-letter" 伪元素：

  • font
  • color
  • background
  • margin
  • padding
  • border
  • text-decoration
  • vertical-align (仅当 float 为 none 时)
  • text-transform
  • line-height
  • float
  • clear

伪元素的优先级

伪元素选择的内容越精细，优先级越高。在下面的例子中，段落的第一个字母将显示为红色，其字体大小为 xx-large。第一行中的其余文本显示为蓝色，并以小型大写字母呈现。段落中的其余文本将以默认字体大小和颜色来显示：

p:first-letter{
  color:#ff0000;
  font-size:xx-large;
}

p:first-line{
  color:#0000ff;
  font-variant:small-caps;
}

:before 伪元素

":before" 伪元素可以在元素的内容前面插入新内容。

下面的例子在每个 <h1> 元素前面插入一幅图片：

h1:before
  {
  content:url(logo.gif);
  }

试一试

:after 伪元素

":after" 伪元素可以在元素的内容之后插入新内容。

下面的例子在每个 <h1> 元素后面插入一幅图片：

h1:after
  {
  content:url(logo.gif);
  }

试一试
