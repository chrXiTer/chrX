CSS3 column-fill 属性

CSS 参考手册

实例

规定如何对列进行填充：

div
{
column-fill:auto;
}

浏览器支持

主流浏览器都不支持 column-fill 属性。

定义和用法

column-fill 属性规定如何填充列（是否进行协调）。

默认值： balance
继承性： no
版本： CSS3
JavaScript 语法： object.style.columnFill="auto"

语法

column-fill: balance|auto;
值 描述
balance 对列进行协调。浏览器应对列长度的差异进行最小化处理。
auto 按顺序对列进行填充，列长度会各有不同。
