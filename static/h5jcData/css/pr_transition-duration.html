CSS3 transition-duration 属性

CSS 参考手册

实例

让过渡效果持续 5 秒：

div
{
transition-duration: 5s;
-moz-transition-duration: 5s; /* Firefox 4 */
-webkit-transition-duration: 5s; /* Safari 和 Chrome */
-o-transition-duration: 5s; /* Opera */
}

亲自试一试

浏览器支持

目前所有浏览器都不支持 transition-duration 属性。

Firefox 4 支持替代的 -moz-transition-duration 属性。

Safari 和 Chrome 支持替代的 -webkit-transition-duration 属性。

Opera 支持替代的 -o-transition-duration 属性。

定义和用法

transition-duration 属性规定完成过渡效果需要花费的时间（以秒或毫秒计）。

默认值： 0
继承性： no
版本： CSS3
JavaScript 语法： object.style.transitionDuration="5s"

语法

transition-duration: time;
值 描述
time

规定完成过渡效果需要花费的时间（以秒或毫秒计）。

默认值是 0，意味着不会有效果。
