CSS width 属性

CSS 参考手册

定义和用法

width 属性设置元素的宽度。

说明

这个属性定义元素内容区的宽度，在内容区外面可以增加内边距、边框和外边距。

行内非替换元素会忽略这个属性。

默认值： auto
继承性： no
版本： CSS1
JavaScript 语法： object.style.width="50px"

实例

设置段落的高度和宽度：

p
  {
  height:100px;
  width:100px;
  }

亲自试一试

可能的值

值 描述
auto 默认值。根据内容增加宽度，根据浏览器宽度换行
length 使用 px、cm 等单位定义宽度。
% 定义基于包含块（父元素）宽度的百分比宽度。
inherit 规定应该从父元素继承 width 属性的值。
