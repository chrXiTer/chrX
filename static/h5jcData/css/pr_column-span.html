实例

使 h2 元素横跨所有列：

h2
{
-webkit-column-span:all; /* Chrome */
column-span:all;
}

浏览器支持

只有 Opera 支持 column-span 属性。

Chrome 支持替代的 -webkit-column-span 属性。

定义和用法

column-span 属性规定元素应横跨多少列。

默认值： 1
继承性： no
版本： CSS3
JavaScript 语法： object.style.columnSpan="all"

语法

column-span: 1|all;
值 描述 测试
1 元素应横跨一列。 测试
all 元素应横跨所有列。 测试
